# Python Programming

This family contains programs about writing Python systems with strong runtime
models, explicit contracts, and maintainable abstractions.

## Included Programs

### Python Object-Oriented Programming

- Location: `programs/python-programming/python-object-oriented-programming`
- Focus: object semantics, responsibilities, state design, aggregates, persistence, verification, and operational evolution

### Python Functional Programming

- Location: `programs/python-programming/python-functional-programming`
- Focus: pure composition, effects, streaming, async, and long-lived refactoring

### Python Meta-Programming

- Location: `programs/python-programming/python-meta-programming`
- Focus: introspection, decorators, descriptors, metaclasses, and runtime semantics
