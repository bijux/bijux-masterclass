# Python Programming

This family contains courses about writing Python systems with strong runtime
models, explicit contracts, and maintainable abstractions.

## Included Courses

### Python Functional Programming

- Location: `courses/python-programming/python-functional-programming`
- Focus: pure composition, effects, streaming, async, and long-lived refactoring

### Python Meta-Programming

- Location: `courses/python-programming/python-meta-programming`
- Focus: introspection, decorators, descriptors, metaclasses, and runtime semantics
