# Library

The library is the synced course and capstone tree published directly from `programs/`.
Use it when you want to browse the checked-in material without the extra routing layer of
the root catalog home page.

## Families

- [Reproducible Research](reproducible-research/index.md)
- [Python Programming](python-programming/index.md)

## How to Use This Library

- Start with a family index when you know the problem space but not the specific course.
- Open a course home when you already know the program you want.
- Use the sidebar to move through the ordered course tree, capstone guide set, and reference pages from one place.
