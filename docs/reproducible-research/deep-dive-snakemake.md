# Deep Dive Snakemake

Deep Dive Snakemake teaches workflow design with explicit inputs and outputs,
safe dynamic behavior, publishable artifacts, and reproducible execution.

## Repository Location

`programs/reproducible-research/deep-dive-snakemake`

## What It Contains

- A course book in `course-book/`
- A reference workflow in `capstone/`
- Local development entrypoints in `Makefile`

## Local Commands

```bash
make PROGRAM=reproducible-research/deep-dive-snakemake program-help
```
