# Deep Dive Make

Deep Dive Make teaches GNU Make as a disciplined dependency-graph engine rather
than as an ad hoc scripting surface.

## Repository Location

`programs/reproducible-research/deep-dive-make`

## What It Contains

- A learner entry route and course navigation pages in `course-book/`
- A ten-module program guide in `course-book/`
- An executable capstone in `capstone/`
- Local development entrypoints in `Makefile`

## Local Commands

```bash
make PROGRAM=reproducible-research/deep-dive-make docs-build
make PROGRAM=reproducible-research/deep-dive-make test
```

## Best First Reads

Start with `course-book/start-here.md`, then `course-book/module-00.md`, then
`course-book/capstone-map.md`.
