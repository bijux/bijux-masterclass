# Deep Dive Make

Deep Dive Make teaches GNU Make as a disciplined dependency-graph engine rather
than as an ad hoc scripting surface.

## Repository Location

`courses/reproducible-research/deep-dive-make`

## What It Contains

- A five-module course book in `course-book/`
- An executable capstone in `capstone/`
- Local development entrypoints in `Makefile`

## Local Commands

```bash
make COURSE=reproducible-research/deep-dive-make docs-build
make COURSE=reproducible-research/deep-dive-make test
```
