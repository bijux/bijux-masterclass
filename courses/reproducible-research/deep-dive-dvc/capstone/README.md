# Deep Dive DVC Capstone

This directory is reserved for the executable reference repository that will accompany the Deep Dive DVC course.

The intended capstone is a compact but realistic DVC project that demonstrates:

- content-addressed datasets and remotes
- truthful `dvc.yaml` stages with stable `dvc.lock` evidence
- declared params and metrics
- disciplined experiment flows
- CI-backed restoration and incident drills

The current import adds the course-book draft first, so this directory starts as a documented placeholder rather than a partial implementation.
